# Archive/ — 归档中心

这里存放 **已完成使命或不再适用** 的历史文档。

## 放什么

- 被新方案替代的旧设计。
- 已经放弃 / 搁置的想法。
- 阶段性完成、不再需要日常参考的内容。

## 原则

- **不删除**，保留下来用于追溯「当初为什么这么决定 / 为什么放弃」。
- 日常执行 **以 `Releases/` 为准**，不要照着 `Archive/` 里的内容做事。
- 从 `Releases/` 移入归档时，记得在 `Releases/` 的新文档里写明替代 / 迁移关系。

## 给 AI agent 的提醒

`Archive/` 只作历史回溯，**不要据此执行**当前工作。
