# Drafts/ — 草稿与上下文素材库

这里存放 **零碎想法、灵感、外部信息，以及与 AI（GPT / Gemini 等）的对话记录**。它是思路的「捕获区」，也是给 AI agent 的上下文背景。

## 放什么

- 任何一闪而过的想法、需求片段、待探索的方案。
- 收集到的外部资料（文章要点、截图要点、链接 + 摘要）。
- 与 GPT / Gemini 等的对话记录（导出为 markdown 直接丢进来）。
- 半成品的梳理、想要做但还没定的事。

## 原则

- **允许混乱**，重在快速捕获，不追求完美。
- 但尽量在开头写清楚：**「我为什么写它、现在卡在哪、下一步要补什么信息」**，方便日后（或 AI）接续。
- 这里的内容 **未定稿、可能不准确**。成熟后移入 `Releases/`；过时后移入 `Archive/`。

## 给 AI agent 的提醒

`Drafts/` 只作参考，不要当成权威结论。冲突时以 `Releases/` 为准。

## 约定

不便公开的私密草稿可放入 `Drafts/private/`（已在 `.gitignore` 中忽略）。
